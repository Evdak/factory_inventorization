# factory_inventorization

Для запуска проекта используйте эти команды (из корневой папки проекта):
Если не установлен pipenv:
<br>
`pip3 install pipenv`

Для запуска:
<br>
`pipenv shell`
<br>
`pipenv install`
<br>
`pipenv run python3 manage.py runserver`
<br>

Сервер запущен по адресу: http://127.0.0.1:8000
